![Dashboard](images/dashboard.png)

## 📌 Project Overview

This project analyzes bike station sharing data using Power BI. It provides insights into station performance, bike availability, contract-wise analysis, and trends over time.

---

## 🎯 Business Problem

Public bike-sharing operators need to monitor station availability to improve customer satisfaction and optimize bike distribution.

---

## ✅ Objectives

- Analyze bike station performance
- Monitor bike availability
- Track trends over time
- Identify low-availability stations
- Build an interactive Power BI dashboard

## 💡 Key Insights

✔ Marseille has the highest number of stations.

✔ Several stations require immediate attention.

✔ Bike availability varies across contracts.

✔ Closed stations reduce overall availability.

## 🚀 Recommendations

- Rebalance bikes

- Monitor closed stations
